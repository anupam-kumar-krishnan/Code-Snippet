# _Code Snippet_
### _Made With_
- React JS
- Tailwind CSS
- Shadcn UI
- Zustand

### _Features_
- Auto Detect Language
- Different Themes
- Number of Fonts
- Select Font Size
- Choose Dark Mode / Light Mode
- Choose Remove / Add Background
- Export as Image/SVG
- Copy Image
- Padding (Resizable)
